# Introduction 

Mark6 utils used by SHAO

## Installation

Just type the following command to install all the utils

```bash
$ git clone https://github.com/shaoguangleo/shao-mark6-utils.git
$ cd shao-mark6-utils
$ mkdir build
$ cd build
$ cmake .. & make & make install
```
